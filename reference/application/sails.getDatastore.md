# sails.getDatastore()
Access the datastore instance.

```javascript
sails.getDatastore(dataStoreName);
```

### Usage


|   |          Argument           | Type                | Details
|---|---------------------------- | ------------------- |:-----------
| 1 |        dataStoreName        | ((string?))         | Defaults to `'default'`.

#### Returns

**Type:** ((Dictionary))

The registered datastore instance (RDI)
